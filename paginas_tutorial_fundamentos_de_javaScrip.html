Introducción a JavaScript

JavaScript es un lenguaje de programación interpretado, orientado a objetos, débilmente tipado y dinámico. Fue creado originalmente por Brendan Eich en Netscape Communications Corporation y se ha convertido en uno de los pilares fundamentales del desarrollo web moderno.

JavaScript permite crear páginas web dinámicas, añadiendo funcionalidades como validación de formularios, animaciones, efectos visuales, interactividad con el usuario y comunicación asincrónica con el servidor. A diferencia de HTML y CSS que se encargan de la estructura y el diseño respectivamente, JavaScript se encarga del comportamiento de la página.

Actualmente, JavaScript no solo se utiliza en el navegador (frontend), sino también en el servidor (Node.js), aplicaciones móviles (React Native), aplicaciones de escritorio (Electron) y muchos otros entornos.

Variables y Tipos de Datos

En JavaScript, las variables son contenedores para almacenar datos. Existen tres formas de declarar variables:

// Usando var (forma antigua)
var nombre = "Juan";

// Usando let (recomendado para variables que cambian)
let edad = 25;

// Usando const (para valores constantes)
const PI = 3.1416;

JavaScript tiene varios tipos de datos fundamentales:

  • String: Para textos. Ejemplo: "Hola mundo"
  • Number: Para números (enteros y decimales). Ejemplo: 42, 3.14
  • Boolean: Para valores verdadero/falso. Ejemplo: true, false
  • Undefined: Para variables declaradas sin valor. Ejemplo: let x;
  • Null: Para representar la ausencia de valor. Ejemplo: let y = null;
  • Object: Para colecciones de datos. Ejemplo: {nombre: "Ana", edad: 30}
  • Array: Para listas ordenadas. Ejemplo: [1, 2, 3, 4]

Operadores en JavaScript

JavaScript incluye varios tipos de operadores que nos permiten realizar diferentes operaciones con valores y variables:

Operadores Aritméticos: Se utilizan para realizar operaciones matemáticas básicas.

let a = 10;
let b = 5;

let suma = a + b; // 15
let resta = a - b; // 5
let multiplicacion = a * b; // 50
let division = a / b; // 2
let modulo = a % b; // 0 (el resto de la división)
let exponente = a ** b; // 100000 (10 elevado a 5)

Operadores de Comparación: Se utilizan para comparar valores y devuelven un valor booleano.

let a = 10;
let b = 5;
let c = "10";

a > b; // true (mayor que)
a < b; // false (menor que)
a >= b; // true (mayor o igual que)
a <= b; // false (menor o igual que)
a == c; // true (igualdad, convierte tipos)
a === c; // false (igualdad estricta, verifica tipo)
a != b; // true (desigualdad)
a !== c; // true (desigualdad estricta)

Operadores Lógicos: Se utilizan para combinar expresiones booleanas.

let x = true;
let y = false;

x && y; // false (AND lógico)
x || y; // true (OR lógico)
!x; // false (NOT lógico)

Estructuras de Control

Las estructuras de control permiten modificar el flujo de ejecución de un programa. En JavaScript, tenemos varias estructuras importantes:

Condicionales: Permiten ejecutar diferentes bloques de código según una condición.

// Estructura if-else
let edad = 18;

if (edad >= 18) {
console.log("Eres mayor de edad");
} else {
console.log("Eres menor de edad");
}

// Estructura if-else if-else
let nota = 85;

if (nota >= 90) {
console.log("Sobresaliente");
} else if (nota >= 70) {
console.log("Notable");
} else if (nota >= 60) {
console.log("Aprobado");
} else {
console.log("Reprobado");
}

// Operador ternario
let resultado = (edad >= 18) ? "Mayor de edad" : "Menor de edad";

Bucles: Permiten ejecutar un bloque de código repetidamente.

// Bucle for
for (let i = 0; i < 5; i++) {
console.log("Iteración " + i);
}

// Bucle while
let contador = 0;
while (contador < 5) {
console.log("Contador: " + contador);
contador++;
}

// Bucle do-while
let num = 1;
do {
console.log("Número: " + num);
num++;
} while (num <= 5);

Switch: Permite evaluar una expresión e igualarla con diferentes casos.

let dia = 3;
let nombreDia;

switch (dia) {
case 1:
nombreDia = "Lunes";
break;
case 2:
nombreDia = "Martes";
break;
case 3:
nombreDia = "Miércoles";
break;
case 4:
nombreDia = "Jueves";
break;
case 5:
nombreDia = "Viernes";
break;
case 6:
case 7:
nombreDia = "Fin de semana";
break;
default:
nombreDia = "Día inválido";
}
console.log("Hoy es " + nombreDia);

Funciones en JavaScript

Las funciones son bloques de código diseñados para realizar una tarea específica. Permiten reutilizar código y organizar mejor nuestros programas.

Declaración de Funciones: Existen varias formas de declarar funciones en JavaScript.

// Declaración de función
function saludar(nombre) {
return "¡Hola, " + nombre + "!";
}

// Expresión de función
const despedir = function(nombre) {
return "¡Adiós, " + nombre + "!";
};

// Función flecha (arrow function)
const sumar = (a, b) => a + b;

Parámetros y Argumentos: Las funciones pueden recibir parámetros y devolver valores.

function calcularArea(base, altura) {
return base * altura;
}

// Llamando a la función
let area = calcularArea(5, 3); // area = 15

Funciones con Parámetros por Defecto: Podemos asignar valores predeterminados a los parámetros.

function saludar(nombre = "amigo") {
return "¡Hola, " + nombre + "!";
}

saludar(); // "¡Hola, amigo!"
saludar("María"); // "¡Hola, María!"

Ámbito (Scope) de las Variables: Las variables declaradas dentro de una función tienen un ámbito local.

let x = 10; // Variable global

function test() {
let y = 20; // Variable local
console.log(x); // 10 (puede acceder a la variable global)
console.log(y); // 20
}

test();
console.log(x); // 10
// console.log(y); // Error: y no está definida fuera de la función
